# company-api-types

Auto-generated TypeScript types for the Redesign Health Company API (OpenAPI v0.0.5-SNAPSHOT). Generated by [OpenAPI Generator](https://openapi-generator.tech).

> **Do not edit this library manually.** Re-generate from the OpenAPI spec when the API contract changes.

## Usage

```ts
import { Company, Vendor, CeoProfile } from '@redesignhealth/company-api-types'
```

## Regenerating Types

Run the OpenAPI Generator against the Company API spec:

```bash
npx openapi-generator-cli generate \
  -i <openapi-spec-url-or-file> \
  -g typescript-axios \
  -o libs/company-api-types/src
```

Then commit the updated files.

nx test company-api-types
