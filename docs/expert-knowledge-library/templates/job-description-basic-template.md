# Job Description Basic Template

Use this basic job description template as a writing guide when creating a job posting or as a brainstorming tool when crafting a role. Note that it's not a full job posting template but focuses in on the job description portion.

This template includes sections for:

- Title, scope, and goals for the role.

- Team and manager for the role.

- What the person in the role is responsible for handling, managing, and achieving.

- What background experience and skills the person in the role needs to be successful.
