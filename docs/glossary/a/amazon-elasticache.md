# Amazon ElastiCache

*Amazon [ElastiCache](https://docs.aws.amazon.com/elasticache/index.html?pg=ln)* is a is a fully managed, in-memory caching service. ElasticCache works with Redis and Memcached engines.

The Platform data infrastructure supports ElasticCache as an option for in-memory caching.
