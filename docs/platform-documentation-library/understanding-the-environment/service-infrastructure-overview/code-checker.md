# Static and Dynamic Code Checkers

## Static Code Chcker - Vericode CE

Veracode Static Analysis provides fast, automated security feedback to developers in the IDE and CI/CD pipeline, conducts a full Policy Scan before deployment, and gives clear guidance on how to find, prioritize, and fix issues fast – helping to significantly scale DevSecOps programs.

**Category:** Static Code

**Website:** [Veracode](https://www.veracode.com/products/binary-static-analysis-sast) CE

**License:** [MIT](https://github.com/veracode/Veracode-Community-Projects/blob/master/LICENSE)

**Documentation:** [Guide](https://docs.veracode.com/)

**Repository:** [GitHub](https://github.com/veracode/Veracode-Community-Projects)


## Dynamic Code Checker - OWASP ZAP

The OWASP Zed Attack Proxy (ZAP) is one of the world’s most popular free security tools and is actively maintained by a dedicated international team of volunteers. It can help you automatically find security vulnerabilities in your web applications while you are developing and testing your applications. It's also a great tool for experienced pentesters to use for manual security testing.

**Category:** Dynamic Code

**Website:** [OWASP ZAP](https://owasp.org/www-project-zap/)

**License:** [Apache License 2.0](https://github.com/zaproxy/zaproxy/blob/main/LICENSE)

**Documentation:** [Guide](https://www.zaproxy.org/docs/)

**Repository:** [GitHub](https://github.com/zaproxy/zaproxy)
