# Shadow Box

Use the `ShadowBox` component to display content in a specifically styled Box, which extends BoxProps.

## Import

```js
import { ShadowBox } from '@redesignhealth/ui'
```

## Usage

Simple card with max-width and copy content.

```jsx
<ShadowBox maxW="500px">
  There are many benefits to a joint design and development system. Not only
  does it bring benefits to the design team, but it also brings benefits to
  engineering teams.
</ShadowBox>
```
