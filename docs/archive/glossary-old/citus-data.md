# Citus Data

*[Citus](https://docs.citusdata.com/en/v11.1/get_started/what_is_citus.html)* is an open-source PostgreSQL extension that transforms PostgreSQL into a distrubuted database. It allows high performance at any scale. Because Citus is an exension, you can leverage Postgres features, tooling, and ecosystem.

The Platform data infrastructure supports Aurora PosgreSQL as an option for a relational database.
